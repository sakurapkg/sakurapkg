# 命令
### 櫻花添加（包）
pyproject.toml 添加需求包（不工作）
### 櫻花安裝（包）
pyproject.toml add requiements package and install package(Adding packages to pyproject.toml is not working.)
### 櫻花初始化
您可以生成盡可能接近 npm init 的 pyproject.toml。
